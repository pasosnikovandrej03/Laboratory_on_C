3.1 Умовний оператор - це конструкція у програмуванні, яка дозволяє виконувати різні операції в залежності від заданої умови. Він дозволяє програмі виконувати альтернативний код, якщо певна умова виконується.

3.2 Є два основних умовних оператори в багатьох мовах програмування:

Оператор if: Використовується для виконання коду, якщо певна умова визначена як істинна (true).

Оператор switch: Дозволяє вибирати один з багатьох можливих шляхів виконання коду в залежності від значення виразу-перемикача.

3.3 Структура оператора switch виглядає так:

cpp
Copy code
switch (вираз-перемикач) {
    case значення-1:
        // Код, який виконується, коли вираз-перемикач дорівнює значенню-1
        break;
    case значення-2:
        // Код, який виконується, коли вираз-перемикач дорівнює значенню-2
        break;
    // інші варіанти значень
    default:
        // Код, який виконується, якщо жоден варіант не відповідає виразу-перемикачу
}
3.4 Форма тернарного оператора (тернарний умовний оператор) виглядає наступним чином:

cpp
Copy code
вираз-умова ? вираз-якщо-true : вираз-якщо-false
Це короткий спосіб обирати між двома виразами на основі умови. Якщо умова істинна, виконується перший вираз (вираз-якщо-true), в іншому випадку виконується другий вираз (вираз-якщо-false).
